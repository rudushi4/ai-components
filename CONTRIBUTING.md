# Contributing to AI Components\n\nThank you for your interest in contributing to AI Components!\n\n## Getting Started\n\n1. Fork the repository\n2. Clone your fork: `git clone https://github.com/YOUR_USERNAME/ai-components.git`\n3. Install dependencies: `pnpm install`\n4. Start development server: `pnpm dev`\n\n## Development Guidelines\n\n### Code Style\n\n- Use TypeScript for all new code\n- Follow the existing code patterns and naming conventions\n- Use Tailwind CSS for styling\n- Ensure components are accessible (WAI-ARIA compliant)\n\n### Component Guidelines\n\nAll components must follow these UI/UX rules:\n\n**MUST:**\n- Hit target >=24px (mobile >=44px)\n- Visible focus rings using `:focus-visible`\n- Full keyboard support per WAI-ARIA APG\n- Honor `prefers-reduced-motion`\n\n**NEVER:**\n- Block paste in inputs\n- Disable browser zoom\n\n### Testing\n\n- Test keyboard navigation\n- Test with screen readers\n- Verify mobile and desktop layouts\n- Check reduced motion behavior\n\n## Pull Request Process\n\n1. Create a feature branch: `git checkout -b feature/your-feature`\n2. Make your changes\n3. Run linting: `pnpm lint`\n4. Build to verify: `pnpm build`\n5. Commit with clear messages\n6. Push and create a Pull Request\n\n## License\n\nBy contributing, you agree that your contributions will be licensed under the MIT License.\n
